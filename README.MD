# IYORA Company Profile

## Deskripsi

IYORA (Indonesian Youth Outstanding Recognition Association) Company Profile Website merupakan website responsif yang dibuat menggunakan HTML, CSS, dan JavaScript untuk menampilkan informasi organisasi, program, event, galeri, tim, dan kontak.

## Fitur

* Responsive Design
* Sticky Navigation Bar
* Hero Section
* About Us Section
* Statistics Counter Animation
* Program Cards
* Event Filter
* Gallery Section
* Lightbox Gallery
* Team Section
* Contact Form Validation
* Google Maps Integration
* Mobile Hamburger Menu

## Teknologi yang Digunakan

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Font Awesome
* Google Maps Embed

## Cara Menjalankan Project

1. Download atau clone repository.
2. Buka folder project menggunakan Visual Studio Code.
3. Install extension Live Server.
4. Klik kanan file `index.html`.
5. Pilih **Open with Live Server**.
6. Website akan berjalan pada browser.
